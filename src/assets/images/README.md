# Images Directory

Please place your images in the following locations:

## Hero Section
- **Graduate Image**: Place your hero/graduate image as `hero/graduate.png`
  - Recommended size: 400x500px or similar aspect ratio
  - Supported formats: PNG, JPG, JPEG, WEBP

## Academic Partners Logos
Place the university logos in the `partners/` folder with these exact filenames:

1. **University of Pennsylvania**: `partners/penn.png`
2. **Harvard University**: `partners/harvard.png`
3. **University of Oxford**: `partners/oxford.png`
4. **Yale University**: `partners/yale.png`
5. **Princeton University**: `partners/princeton.png`

### Logo Requirements:
- Recommended size: 200-300px width
- Transparent background (PNG) preferred
- Supported formats: PNG, JPG, JPEG, SVG, WEBP

If you have different filenames, you can either:
1. Rename your files to match the expected names above, OR
2. Update the import statements in `src/components/AcademicPartners.js` to match your filenames
